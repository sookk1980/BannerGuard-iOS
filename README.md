# BannerGuard - Illegal Banner Reporting iOS App

## Features
- Firebase Auth (Email + SSO)
- Map-based reporting with photo upload
- Report listing and status tracking
- Minimalist SwiftUI design

## Setup
1. Add Firebase via Swift Package Manager
2. Download GoogleService-Info.plist from Firebase Console
3. Enable Auth, Firestore, Storage, Messaging

## Tech Stack
- SwiftUI
- Firebase
- MapKit
